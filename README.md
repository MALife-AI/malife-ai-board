## 핵심 컨셉: AI Hub(상단 탭) + AI Board(좌측 사이드바) 이원 운영

두 채널을 조합하되, **역할을 명확히 분리**합니다.
**AI 혁신팀**이 전체 운영을 총괄하며, **AI Champion 교육**을 중심축으로 두 채널을 연결합니다.

```
┌─────────────────────────────────────────────────────────┐
│  메일 | 게시 | 업무 | 설문 | 제안 | 🔵 AI Hub           │  ← 상단 탭 추가
├──────────┬──────────────────────────────────────────────┤
│ 📌 AI 게시판 │                                          │
│  ├ 성공사례  │       (메인 콘텐츠 영역)                   │
│  ├ 꿀팁공유  │                                          │
│  ├ Q&A     │                                          │
│  └ 아이디어  │                                          │
│ ────────── │                                          │
│ 📁 전사게시  │                                          │  ← 좌측 사이드바
│ 📁 마케팅   │                                          │     AI Board 추가
│ 📁 S/영업   │                                          │
│   ...      │                                          │
└──────────┴──────────────────────────────────────────────┘
```

### 왜 이원 운영인가?

| 단일 채널의 한계 | 이원 운영의 해결 |
|---|---|
| AI Hub만 있으면 직원 참여 공간이 부족 | AI Board가 **직원 참여의 장**을 담당 |
| AI Board만 있으면 콘텐츠가 묻히고 관심 유도 어려움 | AI Hub가 **큐레이션과 동기부여**를 담당 |
| 교육과 커뮤니티가 섞이면 정보 탐색이 어려움 | 채널별 역할 분리로 **목적에 맞는 탐색** 가능 |

## AI Champion 교육 — 두 채널을 잇는 핵심 프로그램

> AI 혁신팀이 추진하는 **AI Champion 교육**은 단순한 교육 과정이 아니라,
> AI Hub와 AI Board를 연결하는 **AX 확산의 엔진** 역할을 합니다.

### AI Champion이란?

각 부서에서 선발된 AI 활용 선도자로, 교육을 이수한 뒤 부서 내 AI 확산을 주도하는 역할입니다.

### AI Champion 교육 과정

| 단계 | 교육 내용 | 연계 채널 |
|---|---|---|
| **Step 1. 기초 교육** | AI 기본 개념, 사내 승인 도구 사용법, 보안 가이드라인 | AI Hub 학습관 |
| **Step 2. 실전 과제** | 본인 업무에 AI를 적용하여 실제 성과 도출 | AI Board 성공사례 작성 |
| **Step 3. 지식 공유** | 학습한 내용과 노하우를 동료에게 전파 | AI Board 꿀팁/Q&A 활동 |
| **Step 4. Champion 인증** | 과정 완료 후 AI Champion 인증 부여 | AI Hub 대시보드에 Champion 소개 |

### AI Champion의 역할 (교육 이수 후)

| 역할 | 활동 내용 | 활동 공간 |
|---|---|---|
| **부서 내 AI 전파자** | 부서원 대상 AI 도구 사용법 안내, 미니 교육 진행 | 오프라인 + AI Board |
| **성공사례 생산자** | AI 적용 사례를 정기적으로 AI Board에 공유 | AI Board 성공사례 |
| **Q&A 답변자** | AI Board Q&A에서 동료 질문에 답변 (Champion 배지 표시) | AI Board Q&A |
| **아이디어 리더** | 부서별 AI 적용 아이디어를 발굴하고 제안 | AI Board 아이디어 제안 |
| **AI 혁신팀 연결고리** | 현장의 니즈와 피드백을 AI 혁신팀에 전달 | AI Hub ↔ 현업 |

## AI Hub (상단 탭) — "쇼케이스 & 교육"

> **역할:** AI 혁신팀이 운영하는 공식 채널. 직원을 끌어들이고, 교육하고, 동기를 부여하는 **입구**

### 운영 주체
**AI 혁신팀** (관리자 중심 운영)

### 구성

| 영역 | 역할 | 상세 내용 |
|---|---|---|
| **AI 대시보드** | 한눈에 보는 AX 현황 | 이달의 우수사례 배너, AI Board 인기글 하이라이트, AI Champion 활동 현황 |
| **AI Champion 교육** | 핵심 교육 프로그램 | Champion 교육 과정 안내, 신청/수료 현황, 기수별 Champion 소개, 교육 일정 |
| **AI 학습관** | 전 직원 대상 교육 | 난이도별 교육 과정 (입문/중급/고급), VOD 강의, 외부 교육 링크 |
| **AI 도구 가이드** | 사내 승인 도구 안내 | 승인된 AI 도구 목록, 사용 가이드, 보안 유의사항, 도구별 활용 사례 연결 |
| **AI 뉴스레터** | 트렌드 및 공지 | AI 업계 동향, 사내 AX 추진 소식, 정책 변경사항 |

### 핵심 기능

- **AI Champion 현황판:** 기수별 Champion 명단, 부서별 배출 현황, Champion 활동 실적
- **이달의 AI Champion:** 가장 활발하게 활동한 Champion을 메인 배너에 노출 + 포상
- **교육 로드맵:** 일반 직원 → AI Champion → 고급 과정으로 이어지는 성장 경로 제공
- **AI Board 연결:** 각 영역에서 관련 AI Board 게시글로 바로 이동하는 링크

## AI Board (좌측 사이드바) — "참여 & 소통"

> **역할:** 직원이 직접 쓰고, 공유하고, 질문하는 **커뮤니티**
> AI Champion이 콘텐츠 생산과 답변의 핵심 축이 됩니다.

### 운영 주체
전 직원 자유 참여 (AI Champion이 활동을 선도, AI 혁신팀이 모니터링)

### 구성

| 게시판 | 역할 | 상세 내용 |
|---|---|---|
| **AI 성공사례** | 실전 경험 공유 | 템플릿 기반 작성 (부서/도구/효과 수치), AI Champion의 실전 과제 결과물도 게시 |
| **AI 꿀팁 공유** | 가벼운 노하우 확산 | 프롬프트 팁, 업무 자동화 스크립트, 단축키 등 자유 형식 공유 |
| **AI Q&A** | 실시간 문의응답 | AI 도구 사용 중 궁금한 점, AI Champion이 우선 답변 (Champion 배지 표시) |
| **AX 아이디어 제안** | 상향식 혁신 | 직원이 직접 AI 적용 아이디어를 제안, 투표로 우선순위 결정 |

### 핵심 기능

- **템플릿 제공:** 성공사례 작성 시 표준 양식 (배경/도구/과정/성과/교훈)
- **태그 시스템:** 부서별, 도구별, 난이도별 태그로 검색 용이
- **Champion 배지:** AI Champion 수료자의 게시글/답변에 배지 표시 → 신뢰도 향상
- **AI Hub 승격:** 좋아요 다수 획득 게시글은 AI Hub 대시보드에 자동 하이라이트

## 두 채널의 역할 분담 요약

| 구분 | AI Hub (상단 탭) | AI Board (좌측 사이드바) |
|---|---|---|
| **성격** | 공식 채널 (Top-down) | 커뮤니티 (Bottom-up) |
| **운영 주체** | AI 혁신팀 (관리자) | 전 직원 (AI Champion 선도) |
| **콘텐츠 방향** | 큐레이션 / 교육 / 안내 | 직접 작성 / 공유 / 토론 |
| **AI Champion 역할** | 교육 이수, 현황 노출, 포상 | 사례 작성, Q&A 답변, 지식 전파 |
| **업데이트 주기** | 주 1~2회 (AI 혁신팀 편집) | 수시 (직원 자율 게시) |
| **핵심 가치** | 동기부여 + 역량 강화 | 자발적 참여 + 지식 확산 |

## 연계 흐름: AI Hub ↔ AI Champion ↔ AI Board

AI Champion 교육이 두 채널을 잇는 **선순환 구조**를 만듭니다.

```
    ┌──────────── AI Hub (쇼케이스) ──────────────┐
    │                                             │
    │  "이달의 AI Champion"   "AI Champion 교육"    │
    │      ▲ 포상/노출             │ 교육 이수      │
    │      │                      ▼               │
    │      │            ┌── AI Champion ──┐       │
    │      │            │  (부서별 선도자)   │       │
    │      │            └───────┬─────────┘       │
    │      │                    │ 실천/전파         │
    │      │                    ▼                  │
    │  ┌───┴──────── AI Board (커뮤니티) ────┐     │
    │  │                                     │     │
    │  │  성공사례 → 꿀팁 → Q&A → 아이디어     │     │
    │  │  (작성)   (공유)  (답변)  (제안)      │     │
    │  │        ↑ 일반 직원도 자유 참여          │     │
    │  └─────────────────────────────────────┘     │
    └─────────────────────────────────────────────┘

    [직원의 여정]
    ① AI Hub에서 AI Champion 교육 신청 / 일반 교육 이수
    ② AI Champion으로 인증 후 업무에 AI 적용
    ③ AI Board에서 성공사례 공유, 동료 질문에 답변
    ④ 우수 Champion이 AI Hub에 노출 → 다른 직원 동기부여
    ⑤ 새로운 직원이 ①부터 반복 → AX 문화 확산
```

## 도입 로드맵

| 단계 | 시기 | 내용 |
|---|---|---|
| **Phase 1** | 1~2주차 | AI Board 4개 게시판 개설 (기존 게시판 구조 활용, 즉시 오픈) |
| **Phase 2** | 3~4주차 | AI Hub 탭 추가 + AI Champion 교육 과정 페이지 구성 |
| **Phase 3** | 5~6주차 | AI Champion 1기 교육 시작, 수료자 AI Board 활동 개시 |
| **Phase 4** | 7~8주차 | 연계 기능 구현 (Champion 배지, 자동 하이라이트, 이달의 Champion) |
| **Phase 5** | 9주차~ | 운영 안정화, AI Champion 2기 모집, 참여율 분석 및 확대 |

> **Phase 1을 먼저 시작하는 이유:** AI Board는 기존 게시판 구조를 그대로 활용하므로 즉시 오픈 가능. AI Champion 교육 시작 전에 게시판을 먼저 열어 직원들의 자발적 참여를 유도하고, Champion 수료자가 활동할 공간을 미리 확보합니다.

## 기대 효과

| 목적 | 담당 채널 | 기대 효과 |
|---|---|---|
| **자발적 AX 참여** | AI Board | 게시판 형태의 낮은 진입 장벽 → 자유로운 참여 유도 |
| **성공사례 공유** | AI Board → AI Hub | Champion이 사례 작성 → AI Hub 승격 → 전사 확산 |
| **AI 교육** | AI Hub | AI Champion 교육 + 전 직원 학습관으로 체계적 역량 강화 |
| **AX 문화 정착** | 양 채널 + AI Champion | 교육(Hub) → Champion 실천(Board) → 인정(Hub) → 확산 선순환 |
| **현업-혁신팀 연결** | AI Champion | Champion이 현장 니즈를 AI 혁신팀에 전달하는 양방향 소통 채널 |
