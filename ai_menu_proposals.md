# Mi-SQUARE AI 메뉴 구성 제안서

## 현재 전사게시판 현황

![현재 Mi-SQUARE 전사게시판](board.png)

현재 Mi-SQUARE 인트라넷은 상단에 **메일 | 게시 | 업무 | 설문 | 제안** 탭이 있고, 좌측 사이드바에 게시판 카테고리(전사/미래에셋생명, 마케팅활동지침, S/영업관련 등)가 나열된 구조입니다.

아래 3가지 안은 이 구조 위에 **AI 전용 메뉴**를 추가하여 다음 목적을 달성하기 위한 제안입니다.

> **목적**
> - 미래에셋생명 사내 직원들의 **AX(AI Transformation)에 대한 자발적 참여** 유도
> - **AI 활용 성공 사례** 공유 및 확산
> - 체계적인 **AI 교육** 제공

---

## 제안 1: "AI Hub" — 통합 포털형

### 배치 위치
상단 탭에 `AI Hub`을 **6번째 탭으로 추가**

```
메일 | 게시 | 업무 | 설문 | 제안 | 🔵 AI Hub
```

### 서브 메뉴 구성

| 서브 메뉴 | 내용 |
|---|---|
| **AI 성공사례** | 부서별 AI 활용 성공 케이스를 카드형 UI로 게시 (좋아요/댓글/북마크 기능) |
| **AI 챌린지** | 월별 미션 제공 (예: "ChatGPT로 업무 자동화하기"), 참여자 랭킹보드 |
| **AI 학습관** | 난이도별 교육 콘텐츠 큐레이션 (입문/중급/고급), 수료 배지 시스템 |
| **AI 문의방** | AI 도입 관련 질의응답, 사내 AI 전문가 매칭 |

### 선정 이유

- 기존 탭과 **동등한 위치**에 배치하여 접근성이 가장 높음
- 직원들이 메일/업무 확인하듯 **자연스럽게 AI 콘텐츠에 노출**
- 탭 하나로 모든 AI 관련 활동을 **통합 관리** 가능 → 운영 효율 극대화
- 향후 기능 확장(AI 도구 연동, 대시보드 등)에 유리

## 제안 2: "AI Board" — 게시판 확장형

### 배치 위치
좌측 사이드바 **게시판 목록 최상단**에 "AI 게시판" 카테고리 그룹 추가

```
📌 AI 게시판          ← 새로 추가
  ├ AI 우수사례 공유
  ├ AI 활용 꿀팁
  ├ AI 교육 공지
  └ AX 아이디어 제안
──────────────────
📁 전사/미래에셋생명   ← 기존
📁 마케팅활동지침
📁 S/영업관련
   ...
```

### 서브 게시판 구성

| 서브 게시판 | 내용 |
|---|---|
| **AI 우수사례 공유** | 성공/실패 사례를 템플릿 기반으로 작성 (부서, 활용 도구, 효과 수치 등) |
| **AI 활용 꿀팁** | 짧은 팁/노하우를 자유롭게 공유, 태그 기반 검색 |
| **AI 교육 공지** | 교육 일정, 자료, VOD 링크 게시 |
| **AX 아이디어 제안** | 직원이 직접 AI 적용 아이디어를 제안하고 투표 |

### 선정 이유

- 기존 Mi-SQUARE UI를 **최소한으로 변경** → 개발 비용 최소
- 직원들이 이미 익숙한 게시판 형태 → **별도 학습 비용 없이** 바로 참여 가능
- 좌측 사이드바 **최상단 배치**로 시인성 확보
- 빠른 도입이 가능하여 **초기 AX 문화 형성에 적합**

## 제안 3: "AI Plaza" — 참여/게이미피케이션형

### 배치 위치
상단 탭 추가 + **메인 대시보드 형태의 별도 페이지** 구성

```
메일 | 게시 | 업무 | 설문 | 제안 | 🏆 AI Plaza
                                      ↓
                    ┌─────────────────────────────────┐
                    │  🏅 이달의 AI 챔피언 (배너)       │
                    ├────────────┬────────────────────┤
                    │ AI 퀘스트   │ 부서 AX 스코어보드   │
                    │ (미션/레벨)  │ (시각화 차트)       │
                    ├────────────┴────────────────────┤
                    │  📚 AI 라이브러리                 │
                    └─────────────────────────────────┘
```

### 주요 영역 구성

| 영역 | 내용 |
|---|---|
| **이달의 AI 챔피언** | 가장 임팩트 있는 AI 활용 사례 선정, 메인 배너에 노출 + 포상 연계 |
| **AI 퀘스트** | 단계별 미션 수행 (교육 이수 → 도구 사용 → 사례 공유 → 동료 교육), 레벨업 시스템 |
| **AI 라이브러리** | 사내 승인 AI 도구 목록, 사용 가이드, 교육 영상 아카이브 |
| **부서 AX 스코어보드** | 부서별 AI 활용 점수를 시각화하여 건전한 경쟁 유도 |

### 선정 이유

- 단순 정보 게시를 넘어 **자발적 참여를 극대화**하는 구조
- 게이미피케이션(미션/레벨/랭킹)은 **내적 동기 자극** → 지속적 참여 유도
- 부서별 스코어보드는 **조직 차원의 AX 확산** 촉진
- 포상 연계 시 참여율 크게 상승
- 경영진에게 AX 추진 현황을 **정량적으로 보고** 가능
