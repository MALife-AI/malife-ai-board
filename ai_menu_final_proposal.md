## 핵심 컨셉: AI Hub(상단 탭) + AI Board(좌측 사이드바) 이원 운영

두 채널을 조합하되, **역할을 명확히 분리**합니다.

```
┌─────────────────────────────────────────────────────────┐
│  메일 | 게시 | 업무 | 설문 | 제안 | 🔵 AI Hub           │  ← 상단 탭 추가
├──────────┬──────────────────────────────────────────────┤
│ 📌 AI 게시판 │                                          │
│  ├ 성공사례  │       (메인 콘텐츠 영역)                   │
│  ├ 꿀팁공유  │                                          │
│  ├ Q&A     │                                          │
│  └ 아이디어  │                                          │
│ ────────── │                                          │
│ 📁 전사게시  │                                          │  ← 좌측 사이드바
│ 📁 마케팅   │                                          │     AI Board 추가
│ 📁 S/영업   │                                          │
│   ...      │                                          │
└──────────┴──────────────────────────────────────────────┘
```

### 왜 이원 운영인가?

| 단일 채널의 한계 | 이원 운영의 해결 |
|---|---|
| AI Hub만 있으면 직원 참여 공간이 부족 | AI Board가 **직원 참여의 장**을 담당 |
| AI Board만 있으면 콘텐츠가 묻히고 관심 유도 어려움 | AI Hub가 **큐레이션과 동기부여**를 담당 |
| 교육과 커뮤니티가 섞이면 정보 탐색이 어려움 | 채널별 역할 분리로 **목적에 맞는 탐색** 가능 |

---

## AI Hub (상단 탭) — "쇼케이스 & 교육"

> **역할:** 회사가 운영하는 공식 채널. 직원을 끌어들이고, 교육하고, 동기를 부여하는 **입구**

### 운영 주체
AX 추진팀 / AI CoE (관리자 중심 운영)

### 구성

| 영역 | 역할 | 상세 내용 |
|---|---|---|
| **AI 대시보드** | 한눈에 보는 AX 현황 | 이달의 우수사례 배너, AI Board 인기글 하이라이트, 교육 참여율 현황 |
| **AI 학습관** | 체계적 교육 제공 | 난이도별 교육 과정 (입문/중급/고급), VOD 강의, 외부 교육 링크, 수료 현황 |
| **AI 도구 가이드** | 사내 승인 도구 안내 | 승인된 AI 도구 목록, 사용 가이드, 보안 유의사항, 도구별 활용 사례 연결 |
| **AI 뉴스레터** | 트렌드 및 공지 | AI 업계 동향, 사내 AX 추진 소식, 정책 변경사항 |

### 핵심 기능

- **이달의 AI 챔피언:** AI Board 성공사례 중 우수 사례를 선정하여 메인 배너에 노출 + 포상
- **교육 로드맵:** 직급/직무별 추천 학습 경로 제공
- **AI Board 연결:** 각 영역에서 관련 AI Board 게시글로 바로 이동하는 링크

---

## AI Board (좌측 사이드바) — "참여 & 소통"

> **역할:** 직원이 직접 쓰고, 공유하고, 질문하는 **커뮤니티**

### 운영 주체
전 직원 (자유 참여, 관리자는 모니터링)

### 구성

| 게시판 | 역할 | 상세 내용 |
|---|---|---|
| **AI 성공사례** | 실전 경험 공유 | 템플릿 기반 작성 (부서/도구/효과 수치), 좋아요/댓글로 피드백 |
| **AI 꿀팁 공유** | 가벼운 노하우 확산 | 프롬프트 팁, 업무 자동화 스크립트, 단축키 등 자유 형식 공유 |
| **AI Q&A** | 실시간 문의응답 | AI 도구 사용 중 궁금한 점, 트러블슈팅, 사내 AI 전문가 답변 |
| **AX 아이디어 제안** | 상향식 혁신 | 직원이 AI 적용 아이디어를 제안, 투표로 우선순위 결정 |

### 핵심 기능

- **템플릿 제공:** 성공사례 작성 시 표준 양식 (배경/도구/과정/성과/교훈)
- **태그 시스템:** 부서별, 도구별, 난이도별 태그로 검색 용이
- **AI Hub 승격:** 좋아요 다수 획득 게시글은 AI Hub 대시보드에 자동 하이라이트

## 두 채널의 역할 분담 요약

| 구분 | AI Hub (상단 탭) | AI Board (좌측 사이드바) |
|---|---|---|
| **성격** | 공식 채널 (Top-down) | 커뮤니티 (Bottom-up) |
| **운영 주체** | AX 추진팀 (관리자) | 전 직원 (자유 참여) |
| **콘텐츠 방향** | 큐레이션 / 교육 / 안내 | 직접 작성 / 공유 / 토론 |
| **업데이트 주기** | 주 1~2회 (관리자 편집) | 수시 (직원 자율 게시) |
| **핵심 가치** | 동기부여 + 역량 강화 | 자발적 참여 + 지식 확산 |
| **비유** | 사내보 / 교육포털 | 사내 커뮤니티 / 게시판 |

## 연계 흐름: AI Hub ↔ AI Board

두 채널은 독립적으로 운영되되, 아래와 같이 **상호 연결**됩니다.

```
    ┌──────────── AI Hub (쇼케이스) ────────────┐
    │                                           │
    │  "이달의 AI 챔피언"     "AI 학습관"          │
    │      ▲ 승격                 │ 교육 이수      │
    │      │                     ▼              │
    │  ┌───┴──────── AI Board (커뮤니티) ───┐    │
    │  │                                    │    │
    │  │  성공사례 → 꿀팁 → Q&A → 아이디어    │    │
    │  │  (작성)   (공유)  (질문)  (제안)     │    │
    │  └────────────────────────────────────┘    │
    └───────────────────────────────────────────┘

    [직원의 여정]
    ① AI Hub에서 교육 이수 / 트렌드 확인
    ② AI Board에서 직접 활용 후 꿀팁 or 사례 작성
    ③ 우수 사례가 AI Hub에 승격되어 전사 노출
    ④ 다른 직원이 이를 보고 ①부터 반복 → 선순환
```

## 도입 로드맵

| 단계 | 시기 | 내용 |
|---|---|---|
| **Phase 1** | 1~2주차 | AI Board 4개 게시판 개설 (기존 게시판 구조 활용, 개발 최소) |
| **Phase 2** | 3~4주차 | AI Hub 탭 추가 + 대시보드 구성 (학습관, 도구 가이드) |
| **Phase 3** | 5~6주차 | 연계 기능 구현 (자동 하이라이트, 이달의 챔피언 선정 프로세스) |
| **Phase 4** | 7주차~ | 운영 안정화, 참여율 분석, 콘텐츠 확장 |

> **Phase 1을 먼저 시작하는 이유:** AI Board는 기존 게시판 구조를 그대로 활용하므로 개발 없이 즉시 오픈 가능. 직원 참여 콘텐츠가 먼저 쌓여야 AI Hub에서 큐레이션할 소재가 생깁니다.

## 기대 효과

| 목적 | 담당 채널 | 기대 효과 |
|---|---|---|
| **자발적 AX 참여** | AI Board | 게시판 형태의 낮은 진입 장벽 → 자유로운 참여 유도 |
| **성공사례 공유** | AI Board → AI Hub | 직원이 작성 → 우수사례 AI Hub 승격 → 전사 확산 |
| **AI 교육** | AI Hub | 체계적 교육 경로 + 도구 가이드로 역량 강화 |
| **AX 문화 정착** | 양 채널 연계 | 교육(Hub) → 실천(Board) → 인정(Hub) 선순환 구조 |
